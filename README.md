# Amazon-Q-Application---SkillGap-Analyzer
SkillGap Analyzer helps users identify gaps between their CV and target job profiles. Upload your resume and job description to receive a skill gap analysis and tailored training recommendations, enabling data-driven career development and improving employability.

# Project Description
SkillGap Analyzer is designed to assist job seekers and professionals in understanding where their skills may be lacking relative to their desired job profiles. By uploading a CV and a      target job profile, users receive:
- A skill gap analysis highlighting missing or underdeveloped skills
- Training recommendations to improve employability
- Actionable insights for career development

# Features
- CV Upload – Users can upload their resumes in PDF or DOCX format.
- Job Profile Upload – Upload a target job description or profile for comparison.
- Skill Gap Analysis – Highlights the skills you have versus the skills required.
- Training Recommendation – Suggests courses or resources to bridge gaps.
- User-Friendly Interface – Clean and intuitive cards for input and output.

# Usage
- Open the application in your browser or via the website. [Website: SkillGap Analyzer App](https://ng0siwgo.chat.qbusiness.us-east-1.on.aws/#/app/ad0cfdc4-d6ca-4523-aa1b-c3eb11367843)
- Upload your CV and the job profile you want to analyze.
- Click Analyze.
- View the Skill Gap Analysis card and Training Recommendations card.
- Use the insights to plan your career development.
